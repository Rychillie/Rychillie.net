Esta é a sua primeira nota. Edite este arquivo ou crie novos arquivos Markdown na pasta `content/pt-BR/notes`.

## Começando

Saga usa um pipeline **Reader → Processor → Writer**:

1. **Readers** analisam seus arquivos de conteúdo (como este arquivo Markdown) e os transformam em itens tipados
2. **Processors** podem transformar itens com lógica customizada
3. **Writers** geram os arquivos HTML de saída
